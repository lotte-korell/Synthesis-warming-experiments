Code for: Synthesis of scale-dependent plant biodiversity change in climate warming experiments

Lotte Korell, Martin Andrzejak, Jonathan M. Chase, W. Stanley Harpole, Tiffany M. Knight. Ecology. & Data https://doi.org/10.6084/m9.figshare.32125000

Four datasets associated with this project are found on Figshare DOI listed above

Data
Compositional data These data summarise compositional turnover between pairs of control and treatment plots. Filename: Temp_comp_data_final
Plot level data These data summarise log response ratios of species richness (LRR S) and effective number of species SPie (LRR_SPie) at the plot scale. Filename: Temp_data_final
Site level data These data summarise log response ratios of species richness (LRR S) and effective number of species SPie (LRR_SPie) at the sitescale. Filename: Temp_data_gamma_final
Climate data MAP and MAT at each site for biome figure. Filename: Biome

Code

Three scripts are provided in this repository.

Div_estimates_Temp_publication.R provided code for the bootstapped estimates of the best modles selected and is the basis for Figures 2-4
Div_mod_Temp_publication.R provides code for model selection for the different levels of diversity metrics at the plot, plot-to-plot and site level.
Div_Figures_Temp_publication.R provides code for the Figures creation
